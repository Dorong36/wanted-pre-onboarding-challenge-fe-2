할 일 ToDo 클래스 활용해 생성하는 함수 할 일을 추가할 수 있다. 내용없이 추가할 수 없다.
Parameters:
Name Type Description
할일들 Array.<Object> 생성된 할 일들이 들어갈 Object 배열
아이디 number 생성자에 넣어줄 아이디
내용 string 생성자에 넣어줄 내용 (undefined시 생성 불가)
완료여부 boolean 생성자에 넣어줄 완료여부
카테고리 string 생성자에 넣어줄 카테고리
태그들 Array.<string> 생성자에 넣어줄 태그들

DELETE(할일들, 모두제거, 아이디, 태그모두제거, 태그)

삭제 함수 ID를 기반으로 특정 할 일을 삭제할 수 있다. 모든 할 일을 제거할 수 있다. 특정 할 일의 특정 태그를 삭제할 수 있다. 특정 할 일의 모든 태그를 제거할 수 있다.
Parameters:
Name Type Description
할일들 Array.<Object> 할 일들이 모두 담긴 객체 배열
모두제거 boolean 모든 할 일 제거 여부
아이디 number 제거하고 싶은 할 일의 ID
태그모두제거 boolean 특정 ID 할 일의 태그 모두 제거 여부
태그 string 특정 ID 할 일의 태그 중 제거하고 싶은 태그

업데이트 함수 ID를 제외한 모든 속성을 수정할 수 있다. 특정 할 일의 특정 태그를 수정할 수 있다.
Parameters:
Name Type Description
할일들 Array.<Object> 할 일들이 모두 담긴 객체 배열

항목 string 수정하고자 하는 항목명
새값 any 수정하고자 하는 새 값 (항목에 따라서 자료형 변동, 자료형은 Todo 클래스 참고)
